Summary Description:

This application is a back office application prepared to allow the administrators to keep track of the classes, subjects, students and teachers of a primary school
There are two administrators with the following passwords: admin1 pass1, admin2 pass2. The administrators and passwords cannot be changed by this application.
With the application, the administrators can;
- add, delete, amend and query the records
- assign subjects to classes
- assign teachers to subjects, assigned to classes
- get a master list of students
- view class report, which shows all information on the students, subjects and teachers assigned to the class
